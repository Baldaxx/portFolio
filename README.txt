Portfolio de Virginie Baldacchino
=================================
Ce portfolio numérique dans le cadre de ma formation en développement web et web mobile. 
Il présente une collection de travaux et projets réalisés, illustrant mes compétences et mon parcours dans le domaine du développement web.

Fonctionnalités
--------------
Design Responsive : Conçu pour s'adapter à tous les types d'appareils, garantissant une expérience utilisateur optimale sur mobiles, tablettes et ordinateurs.
Technologies Utilisées : HTML5, CSS3 avec Bootstrap pour le stylisme, JavaScript pour l'interactivité, ainsi que l'utilisation des icônes FontAwesome pour une interface utilisateur élégante.
Interactive UI : Des animations subtiles enrichissent l'expérience utilisateur, grâce à l'utilisation d'Anime.js pour les effets visuels et de scripts personnalisés pour des interactions dynamiques.
Section Projets : Présente une sélection de projets développés, chacun accompagné de descriptions, technologies utilisées, et liens vers le code source ou la démo en direct.
Contact Form : Un formulaire de contact intégré, facilitant la communication avec les visiteurs intéressés ou les recruteurs potentiels.
